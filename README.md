PhilanthroMe: Your giving profile on the web.

## To Run Locally

- Add `.env.local` file to root of project with
```
REACT_APP_FIREBASE_API_KEY=[key goes here]
```
- `npm start`

## To Run Tests
- Add `.env.test.local` file to root of project with
```
REACT_APP_FIREBASE_API_KEY=[key goes here]
```
